# Tanks-Arena

Its a simple console game, where you command tank.
Game target is survive and destroy all other tanks.
Program created with used OOP and patterns.

## Game control
* W - tank moves up
* A - tank moves left
* S - tank moves down
* D - tank moves right
* Space - tank shoots
* P - game pause
* Esc - exit
